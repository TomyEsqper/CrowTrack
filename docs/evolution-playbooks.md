# Evolution Playbooks

- Agregar módulo nuevo: seguir guía, ADR y ownership
- Agregar use case: DTOs, guards, métricas y auditoría
- Agregar tenant grande: límites, backpressure y caché multi-tenant
- Migrar estrategia de datos: ADR, compatibilidad y doble escritura temporal
- Extraer microservicio: puertos, contratos y outbox
